The Fibonacci Sequence is fairly simple. First, you start with the number 1. Then, you progress, adding the previous number to the current number to get a new number. 1 + (null) = 1, 1 + 1 = 2, 2 + 1 = 3, 3 + 2 = 5, and so on.
This program prompts the user with how many spaces they want to go to, and then prints the results.

The Collatz Conjecture is a theory that states that for any positive integer, the number 1 can be reached through a simple logical process:

  • If the number is even, it's divided by 2.
  • If the number is odd, it's multiplied by 3, then one is added
If the process is repeated enough, the number will eventually reach zero.
This program takes a positive integer input from the user, and runs the Collatz Conjecture on the number. It prints the number at each step, and at the end tells you how many steps it took to reach one. If the program doesn't work, try to remember what number you entered, because you may have disproved the Collatz Conjecture (it might be my code not working, but what are the chances of that?)
